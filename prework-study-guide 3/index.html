Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • When you create an opening tag, it automatically creates the closing tag for you
  • You should create section elements for each topic, or thematic content, so the content is organised and users can easily find what they need
  • you can create as many list items and lists as you want using ul for un-ordered lists such as bullet points and ol for numbered lists
  • You should use heading tags h1 to h6 which are titles and subtitles tokeep the page organized
  • using a p tag represents a paragraph or block of text on a page
  • using the img tag you can insert images and attribute those to the source using src
  • The br tag creates a space between two other elements or two blocks of content
  • If you want to insert a link onto the webpage you can do so using the a element which stands for anchor
  • Footer elements appear at the bottom of a webpage and usually contain the author, copyrights,contact, sitemap and navigation

CSS

  • CSS stands for Cascading Style Sheets and is used for styling a webpage
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside the element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git revert: restores another version - versions are labelled using SHA
  • Git desktop allows you to work locally on your computer
  • all git commands start with git
  • git is a version control systemsthat lets you manage and keep track of each version of your project
  • we access git using commandline or CLI
  • You can use git commands to create a repo, add code to a repo and view different versions
  • Use the git cheat sheet to see and learn all commands to carry out tasks such as set up, setup & init, stage & snapshot, Branch & merge and more
  • git and github are not the same thing

JavaScript

  • Javascript is used to add functionality to a webpage
